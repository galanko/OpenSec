style

Design system notes

Every frame complies with the Serene Sentinel rules: no 1px solid borders, tonal layering via Level 0/1/2 surfaces, ghost borders only, sentence case throughout, on-surface for text (never #000), Manrope headlines, Inter body, and Material Symbols Outlined icons. Severity uses error/tertiary/on-surface-variant tokens — never arbitrary greens or reds.
